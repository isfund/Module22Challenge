# Module22Challenge

Project involves analyzing home sales data using Apache Spark in Google Colab. 

The analysis includes:
- Load data from AWS S3 bucket.
- Performed queries on the data to calculate metrics such as average prices based on specific criterias.
- Partitioned the data for optimized storage and querying.
- Compared the performance of cached and non-cached operations.
- Results saved in Parquet format.
